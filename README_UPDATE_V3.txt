GAMESCO ESCAPE QUIZ - UPDATE V3

Gefixt:
- Code-Eingabefelder werden nicht mehr jede Sekunde neu aufgebaut.
- Fake-Seiten beim Bombenrätsel bleiben offen und klappen nicht mehr sofort zu.
- Finale hat eine rote SYSTEMFEHLER-Warnung: Team-Modus beendet.
- Jeder Spieler muss die Warnung bestätigen.
- Erst wenn alle verbundenen Spieler bestätigt haben, startet das Finale-Memory.
- Bombenrätsel hat jetzt einen eigenen Timer.
- Standard-Bombenzeit: 5 Minuten.
- Bei falschem Bomben-Kabel werden 20 Sekunden von der Bombenzeit abgezogen.
- Bombenzeit und Bomben-Fehlerabzug kann der Admin im Bombenbereich ändern.

Start:
1. ZIP entpacken
2. Terminal im Ordner öffnen
3. npm install
4. npm start
5. Admin: http://localhost:3000/admin.html
6. Spieler: http://localhost:3000/player.html
